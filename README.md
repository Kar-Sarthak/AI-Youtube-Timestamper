AI YouTube TimeStamper


AI YouTube TimeStamper enhances the YouTube viewing experience by inserting an interactive timestamp panel directly into the YouTube interface. It retrieves video transcripts, sends them to a cloud backend, and returns structured timestamps that summarize the video into easy-to-navigate topics.

This project combines:

A Chrome extension (JavaScript)

A serverless AWS backend (Lambda + API Gateway)

A Python LangChain pipeline using Gemini API
